# 🌐 Cisco Packet Tracer Network Project

## 📌 Project Overview

This project is a network simulation created using **Cisco Packet Tracer**. It demonstrates fundamental networking concepts such as IP addressing, routing, switching, and device connectivity.

The goal of this project is to design and configure a functional network environment that simulates a real-world enterprise setup.

---

## 🛠️ Tools & Technologies Used

* Cisco Packet Tracer
* Networking Protocols (TCP/IP)
* Routing & Switching Concepts
* IP Addressing & Subnetting

---

## 🧩 Network Features

* 📡 Router configuration
* 🔀 Switch configuration
* 🌐 End device connectivity (PCs/Laptops)
* 🧮 IP Address assignment
* 🔁 Routing between networks
* 📶 Network topology design

---

## ⚙️ Configurations Performed

* Assigned IP addresses to all devices
* Configured routers for inter-network communication
* Set up switches for local network connectivity
* Verified connectivity using **ping commands**
* Ensured proper data flow between different networks

---

## 🧪 Testing & Verification

* ✔️ Successful ping between devices
* ✔️ End-to-end connectivity established
* ✔️ Network functioning as expected
